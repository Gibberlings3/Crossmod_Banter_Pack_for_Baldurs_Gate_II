Overview

This mod will add new material for between various Baldur's Gate II mods including banters, romance conflicts and interjections.

Compatibility

This mod is designed to work with Baldur's Gate II: Shadows of Amn with or without the Throne of Bhaal expansion pack, Baldur's Gate Trilogy, Baldur's Gate II: Enhanced Edition, or Enhanced Edition Trilogy. These components will only add new material to your install if two or more of the mods covered by the project are installed. It will not be able to see into the future and detect something not installed before it looks at your installation - please place it in install order after all of the mods in this list.

Bug Reporting

If you should encounter any bugs, please report them to the authors at the Crossmod Banter Pack forum. In addition, Crossmod Banter Pack is available on GitHub, so fixes and changes can be submitted by the community. All components will generate a log in weidu_external/crossmodbg2/logs which may be useful in debugging.

Installation

First time installing a mod? Check out G3's comprehensive tutorial: A New Player’s Guide to Installing and Playing Mods.

Enhanced Editions Note
The Enhanced Editions are actively supported games. Please note that every patch update will wipe your current mod setup! If in the middle of a modded game you might want to delay the patch update (if possible) as even after reinstalling the mods, you might not be able to continue with your old savegames. Alternatively, copy the whole game's folder into a new one that can be modded and will stay untouched by game patches. For BG:EE it is important that you install the mod to the language version you are playing the game in. Otherwise, the dialogues of the mod will not show but give error messages.

Windows
The Crossmod Banter Pack for Baldur's Gate II for Windows is distributed as a self-extracting archive and includes a WeiDU installer. To install, simply double-click the archive and follow the instructions on screen.

Alternatively, the files can be extracted into your game directory using 7zip or WinRAR. When properly extracted, your game directory will contain setup-crossmodbg2.exe and the folder crossmodbg2. To install, double-click setup-crossmodbg2.exe and follow the instructions on screen.

You can run setup-crossmodbg2.exe in your game folder to reinstall, uninstall or otherwise change components.

macOS
The Crossmod Banter Pack for Baldur's Gate II for macOS is distributed as a compressed tarball and includes a WeiDU installer.

First, extract the files from the tarball into your game directory. When properly extracted, your game directory will contain setup-crossmodbg2, setup-crossmodbg2.command, and the folder crossmodbg2. To install, double-click setup-crossmodbg2.command and follow the instructions on screen.

You can run setup-crossmodbg2.command in your game folder to reinstall, uninstall or otherwise change components.

Linux
The Crossmod Banter Pack for Baldur's Gate II for Linux is distributed as a compressed tarball and does not include a WeiDU installer. Linux users will need to do a one-time install of WeiDU (and a few other adjustments) as described in this great writeup.

To install, run 'WeInstall crossmodbg2 in your game folder.

Note for Complete Uninstallation
In addition to the methods above for removing individual components, you can completely uninstall the mod using setup-crossmodbg2 --uninstall at the command line to remove all components without wading through prompts.

Contents

There are three components to this mod: The SoA content, the ToB content, and the Romance Conflict (Kill Other Romances When One Is Active).

Crossmod Banter Pack for Shadows of Amn

This component (0) will add new material to your install of SoA based on what mods you have installed. The list of mods with Crossmod content are listed at the end of this section.

Crossmod Banter Pack for Throne of Bhaal

This component (1) will add new material to your install of ToB based on what mods you have installed. This component requires the previous component to be installed first. Note, some of these banters may also appear in SoA and therefore if they have all already played in SoA, you will see no new banters in ToB between those characters. The list of mods with Crossmod content are listed at the end of this section.

Crossmod Romance Conflicts

This component (2) will add romance conflicts between many mod NPCs in SoA (and ToB if it is installed). If you prefer multi-romances, don't install this.

Contains:

  • Romance auto-kills for jastey's Ajantis, Amber, Angelo, Chloe, Corthala Romantique, Darian, Fade, Gavin, Keldorn Romance, Kelsey, Imoen Romance, Nalia (via Nalia Romance or de'Arnise Romance), Nathaniel, Neh'taniel, Ninde, Saerileth, Sarah, Sime from Tortured Souls, Weimer's Solaufein, Tashia, Tsujatha, Tyris Flare, and Kulyok's Xan.

Hidden Option for Non-English Players

Near the beginning of the setup-crossmodbg2.tp2 (inside the crossmodbg2 folder, you can open it with any text editor) is the line

OUTER_SET strict_language_enforcement = 0

By default, Crossmod will use any translated files it can find, and then use English if there are no translations. E.g. if you install in German and you have Gavin, Ajantis, and Xan installed, it will use the German translation for Gavin-Ajantis banters but English for Ajantis-Xan banters because there's currently no German translation for the latter. If you change the variable to

OUTER_SET strict_language_enforcement = 1

it will restrict installation exclusively to banters that are entirely available in your language. Using the previous example with Gavin, Ajantis, and Xan: with this change Gavin-Ajantis banters would be installed but the Ajantis-Xan banters would be skipped since there's no German translation.
